
## 🧩 Features Implemented

### 🔹 UI Components
- Responsive Navbar
- Hero Section
- Carousel
- Cards (Services)
- Modal Popup
- Contact Form
- Footer with social icons

### 🔹 Advanced Features
- 🌙 Dark Mode Toggle
- ✨ Scroll Animations (AOS)
- 💎 Modern UI (Glassmorphism + Gradients)
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 📩 Working Contact Form (Formspree Integration)

---

## 📱 Responsiveness
The website is fully responsive across:
- 📱 Mobile devices
- 📲 Tablets
- 💻 Desktop screens

Bootstrap grid system and utility classes were used to ensure adaptability.

## 🌐 Live Demo
🔗 Live Website: https://akshay-kumar12307.github.io/bootstrap/  
🔗 GitHub Repository: https://github.com/akshay-kumar12307/bootstrap

## ⚙️ How to Run Locally
1. Download or clone the repository
2. Open project folder
3. Run `index.html` in browser

---

## ✍️ Project Reflection

### 🧠 Steps Followed
- Studied Bootstrap 5 documentation and examples
- Selected key UI components (navbar, cards, carousel, forms, modal)
- Designed and structured 3 web pages
- Enhanced UI with animations and styling
- Made the website fully responsive
- Integrated contact form using Formspree
- Deployed project using GitHub Pages

---

### 🤖 Tools Used
- Bootstrap Documentation
- ChatGPT (for guidance and improvements)
- GitHub (for hosting and deployment)

---

### ⚠️ Challenges Faced
- Maintaining responsive layout across devices
- Aligning components properly
- Integrating working contact form

---

### ✅ Solutions
- Used Bootstrap grid system effectively
- Applied spacing utilities and media queries
- Used Formspree for backend-less form handling

---

### 📈 Learning Outcomes
- Strong understanding of Bootstrap 5 components
- Improved UI/UX design skills
- Learned responsive web design
- Gained experience in deploying projects

---

### ⏱️ Time Taken
Approximately 2–3 days

---

## 🏆 Conclusion
This project helped in gaining hands-on experience with Bootstrap 5 and building modern, responsive web pages. It also improved design thinking and practical web development skills.
